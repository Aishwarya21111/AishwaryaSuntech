The Product Management System is a robust web application developed using Spring Boot and Thymeleaf to facilitate efficient product management. This project includes a comprehensive backend built with Spring Boot, leveraging JPA for seamless database interactions with a MySQL database. The core entity in the system is the Product, characterized by fields such as prodId, prodName, price, prodtype, imageFileName, and brand.

The application offers full CRUD (Create, Read, Update, Delete) functionality through RESTful endpoints in the ProductController. Users can manage products by performing operations such as adding new products, retrieving product details, updating existing products, and deleting products.

A key feature of the system is the recommendation engine, designed to enhance user experience by suggesting products. The recommendation logic prioritizes the cheapest products of the same brand, returning up to three recommendations. If fewer than three products of the same brand are available, the system supplements the recommendations with other products of the same type.

The frontend of the application is built using Thymeleaf, ensuring seamless integration with the Spring Boot backend. The user interface includes pages for viewing, adding, updating, and deleting products, as well as a dedicated page for product recommendations. Bootstrap is employed to ensure a responsive and visually appealing design, with custom CSS for additional styling.

This project aims to provide a streamlined and user-friendly solution for product management, suitable for businesses looking to manage their inventory effectively. The combination of a powerful backend, intuitive frontend, and advanced recommendation system makes this application a valuable tool for any product-centric business.
